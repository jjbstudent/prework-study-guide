Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The Title displays the name of the page
  • html opening tag, which represents the root, or start, of our HTML document
  • html lang="en" tells the browser the language for the web page. (attribute)

CSS

  • The box-shadow CSS property adds shadow effects around an element's frame
  • Font size and text align will format the text size and position it accordingly
  • background-color blue is used for this border
  • The margin property assigns 40px to the top and bottom, and the auto value to the left and right.
  • The .card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A adds all changed
  • git commit -m "your notes" commits with notes
  • Git push origin "location"- syncs local branch with remote branch

JavaScript

  • An IF statement can be used to setup several conditions
  • console.log can be opened via the browser by right-clicking -inspect
  • An array is a single variable that is used to hold a group of data.
  • JavaScript function is a block of code designed to perform a particular task.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array
